# WinampGera Roadmap

## Now Shipping

- Recent tracks list inside the main player window.
- Automatic restore of the last opened track on launch.
- One-tap sharing for the current track.
- Lightweight local metrics and state storage.

## Next Up

- Queue or playlist support so listening can continue across multiple tracks.
- Richer track metadata like artist, title, and duration.
- Playback progress with seek controls.
- Keyboard shortcuts for open, play/pause, stop, share, and recent reopen.
- Packaging for macOS, Windows, and Linux beta builds.

## Product Bets

- Export a "Now Playing" image card for stories and social posts.
- Smart recommendations from recent listening habits.
- Drag-and-drop file loading to reduce friction.
- Mini-player mode for persistent desktop listening.

## Engineering Focus

- Split the single-file app into playback, persistence, analytics, and UI modules.
- Add higher-confidence tests for clipboard, preferences, and startup restore flows.
- Improve VLC dependency diagnostics across operating systems.
